# Minecraft server - 1.5.2 - Better Than Wolf

## Build the docker image

```sh
docker build --tag clodyx/btw-server:latest .
```

## Run in Docker

```sh
docker run -ti --name btw -p 25565:25565  clodyx/btw-server:latest
```
